# Diagnostic AI Healthcare

An AI-powered healthcare assistant for symptom analysis, report generation, and location-based doctor recommendation.

Features
- Symptom analysis using AI
- Emergency condition detection
- Health report generation
- Doctor recommendation based on location
- WhatsApp report sharing
- Hindi language support

Technologies Used
- HTML
- CSS
- JavaScript
- AI-assisted development

How to Run
1. Download or clone the repository
2. Open index.html in your browser
